## Installation

1. Create a virtual environment.
```bash
python -m venv venv
```

2. Active the virtual environment:
- Windows
````bash
venv\Scripts\activate
````
- MacOS/Linux
````bash
source venv/bin/activate
````

3. Install required packages:
````bash
pip install -r requirements.txt
````

## Run application
````bash
python app.py
````
